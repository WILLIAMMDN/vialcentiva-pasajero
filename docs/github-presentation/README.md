# Presentación GitHub - VialCentiva Pasajero

Este directorio contiene una presentación HTML autocontenida para explicar el proyecto en modo pitch profesional.

## Cómo usarla

Abre `index.html` en un navegador. La presentación usa assets locales del repositorio:

- `../../public/logo_vialcentiva.png`
- `../../.github/readme/hero-vialcentiva.svg`
- `../../.github/readme/mobile-showcase.svg`
- `../../.github/readme/architecture-map.svg`

## Navegación

- Flecha derecha o barra espaciadora: siguiente slide.
- Flecha izquierda: slide anterior.
- Botones inferiores: navegación manual.

## Publicación sugerida

Puedes subir el repositorio a GitHub y activar GitHub Pages apuntando a la carpeta `docs/`. Si prefieres publicar solo esta presentación, conserva la estructura relativa de carpetas para que los assets carguen correctamente.
